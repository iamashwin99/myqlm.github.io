.. module:: qat.plugins.sequential_optimization

Sequential optimization plugin
==============================
This plugin allows for a local tuning of each parameter of the ansatz circuit one after the other, going sequentially through all of them. The notebook :notebook:`(Optimizing circuits with the sequential optimization plugin)[sequential_optimization_plugin.ipynb]` allows to get familiar with the plugin.
