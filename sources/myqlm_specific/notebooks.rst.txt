.. _myqlm-notebooks:

Notebooks
=========
myQLM comes with interactive tutorials you can open with Jupyter Notebook. You
can:

 - Run these notebooks in your browser using
   `myBinder <https://mybinder.org/v2/gh/myQLM/myqlm-notebooks/master?filepath=overview.ipynb>`_
 - Clone the `GitHub repository myqlm-notebooks <https://github.com/myQLM/myqlm-notebooks>`_
   containing all our notebooks
 - Download `a zip file <https://myqlm.github.io/distrib/myqlm_notebooks.zip>`_
   containing all our notebooks

Please start the notebook :code:`overview.ipynb`, this notebook contains a table
of content of all our tutorials.


.. code-block:: text

    jupyter notebook path/to/overview.ipynb
