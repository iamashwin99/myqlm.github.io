Contributing to myQLM
=====================

If you want to share your code on myQLM or add new features to this project,
you can be contributor of our
`myqlm-contrib library <https://github.com/myQLM/myqlm-contrib>`_.

How to contribute?
------------------
To contribute to this project, you have to

 - Create a fork of myqlm-contrib
 - Commit and push your contribution
 - Create a pull request
