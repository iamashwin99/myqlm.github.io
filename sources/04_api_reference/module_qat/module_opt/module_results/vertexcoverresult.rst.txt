qat.opt.results.VertexCoverResult
=================================

The :class:`~qat.generators.VertexCoverGenerator` can return a parsed result that includes a parsed 
version of the best :class:`~qat.core.Sample`, that could be displayed. The node subset that covers all the nodes
in the graph can be accessed through the cover function.

.. autoclass:: qat.opt.results.VertexCoverResult
    :members: cover, display
