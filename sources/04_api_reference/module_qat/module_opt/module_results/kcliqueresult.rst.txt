qat.opt.results.KCliqueResult
=============================

The :class:`~qat.generators.KCliqueGenerator` can return a parsed result that includes a parsed 
version of the best :class:`~qat.core.Sample`, that could be displayed. The nodes that forms the
clique can be accessed through the clique function.
