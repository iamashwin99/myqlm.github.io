qat.comm
========

.. parseonly:: qlm distributed

    This module provides all the data structures needed by Qaptiva. Most objects in this module are the low-level structures that
    constitute the base of the framework. For Qaptiva users, they are mostly required when creating a user-defined custom objects,
    such as a QPU or a Plugin.
