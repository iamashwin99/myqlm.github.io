SeqOptim: optimizing circuits with the sequential optimization algorithm
########################################################################

This plugin allows for a local tuning of each parameter of the ansatz circuit one after the other, going sequentially through all of
them. The notebook :notebook:`(Optimizing circuits with the sequential optimization plugin)[qat_fermion_sequential_optimization_plugin.ipynb]`
allows you to familiarize yourself with the plugin.
