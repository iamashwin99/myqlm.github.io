Plugins for variational algorithmsÂ¶

The following plugins inherits from the class Optimizer and provide an easy way to include your favorite classical optimizer inside a QLM stack. These plugins do not provide particular interface for gradient based optimization.

They require the input job to have abstract parameters (i.e the circuit and/or the observable should be parametrized).

We can, for instance, build a very simple variational job as follows:

from qat.plugins import ScipyMinimizePlugin
from qat.qpus import get_default_qpu
from qat.lang.AQASM import Program, RY
from qat.core import Observable

prog = Program()
qbits = prog.qalloc(1)
RY(prog.new_var(float, "a"))(qbits)
job = prog.to_circ().to_job(observable=Observable.sigma_z(0, 1))

stack = ScipyMinimizePlugin(method="COBYLA") | get_default_qpu()

result = stack.submit(job)

print("Final energy:", result.value)
print("Minimization trace:", result.meta_data["optimization_trace"])
print("Best parameters:", result.meta_data["parameters"])

Scipy minimize bindersÂ¶

The following classes are wrappers for the scipy.optimize.minimize method. Their usage is pretty close to scipy API.

class qat.vsolve.optimize.scipy_binder.ScipyMinimizePluginÂ¶

A particularization of the Optimizer class for the scipy.optimize.minimize method.

Parameters

  • binding_args (dict) â arguments passed to the variable binding method

  • kwargs â minimize parameters (see scipyâs API)

Note

scipy.minimize requires some initialization value for the optimization (parameter x0). In our wrapper, if no value is given, we assume random initilization:

x0 = np.random.random(len(variables))

Parameters can be changed dynamically using the set_options() method.

This plugin is as ready-to-use as it gets:

import numpy as np
form qat.plugins import ScipyMinimizePlugin
from qat.linalg import LinAlg

plugin = ScipyMinimizePlugin(x0=np.random.random(10),
                             method="COBYLA",
                             tol=1e-3,
                             options={"maxiter": 300})

stack = plugin | LinAlg()
# Any job will run, as long as it contains 10 variables!
result = stack.submit(my_job)
